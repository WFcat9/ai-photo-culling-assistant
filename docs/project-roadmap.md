# AI 摄影后期助手项目路线图

## 项目定位

先做一个摄影师每天能用的工具：批量上传照片，自动筛出明显问题照片，给出保留、待修、淘汰建议，再逐步升级为 AI 摄影后期助手。

## 工作树推进方式

主仓库只保留稳定版本。每个阶段单独开一个 Git 工作树，做完、测试通过、确认效果后再合并回主线。

## 阶段 1：批量筛片助手

工作树：`.worktrees/phase1-screening`

目标：
- 批量上传照片
- 显示缩略图列表
- 给每张照片打上“保留 / 待修 / 淘汰”
- 先用浏览器能直接计算的指标做初筛：清晰度、曝光、画面倾斜、尺寸信息
- 做出一个老师和摄影师都能看懂的结果界面

第一阶段不做：
- 不接外部 AI
- 不做人脸库
- 不做自动修脸
- 不追求判断 100% 准，只先做实用筛查

## 阶段 2：水印与导出

工作树：`.worktrees/phase2-watermark-export`

目标：
- 批量加文字水印或图片水印
- 设置位置、透明度、大小
- 批量改尺寸、压缩、导出
- 支持只导出“保留”的照片

## 阶段 3：修图建议

工作树：`.worktrees/phase3-editing-advice`

目标：
- 根据曝光、色温、对比度、饱和度给出基础修图建议
- 给出裁切方向建议
- 输出适合 Lightroom / Photoshop 操作的大白话建议

## 阶段 4：人像与脸型分析实验

工作树：`.worktrees/phase4-face-analysis-lab`

目标：
- 检测人脸位置、闭眼风险、脸部清晰度
- 建立脸型分类规则原型
- 给出谨慎的修脸建议

注意：
- 这一阶段涉及人脸隐私和审美主观性，只做本地实验，不上传照片。

## 阶段 5：多 AI 审美工作流

工作树：`.worktrees/phase5-ai-style-workflow`

目标：
- 把参考图、色彩风格、排版方向整理成工作流
- 支持把 Gemini / 其他 AI 的审美分析结果导入项目
- 最后再考虑动画、视频和内容排版衔接

## 当前优先级

现在只进入阶段 1。阶段 1 完成后，再决定是否创建新对话继续阶段 2。
