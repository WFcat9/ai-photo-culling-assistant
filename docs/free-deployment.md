# 免费部署方案

## 推荐方案：Cloudflare Pages

适合这个项目，因为它是纯前端静态网页，不需要服务器，也不需要付费 API。

部署思路：

1. 把项目推到 GitHub 仓库。
2. 登录 Cloudflare Pages。
3. 选择 GitHub 仓库。
4. 构建命令填写：`npm run build`
5. 输出目录填写：`dist`
6. 部署完成后，把生成的网址发给别人即可使用。

优点：

- 免费额度够个人项目使用。
- 访问速度通常比较稳定。
- 适合 Vite 这种静态前端项目。
- 照片在用户浏览器本地分析，不上传到服务器。

## 备选方案：Netlify

配置同样简单：

- 构建命令：`npm run build`
- 输出目录：`dist`

适合快速分享原型。

## 备选方案：GitHub Pages

也可以免费部署，但如果仓库不是根域名站点，可能需要额外配置 Vite 的 `base` 路径。后续真正发布前再根据仓库名处理。

## 当前项目的免费部署边界

当前版本可以免费部署，因为：

- 不调用 Gemini、OpenAI 或其他付费视觉 API。
- 不需要后端数据库。
- 不上传用户照片。
- 画面分析在浏览器本地完成。

如果未来要加入更强的人脸关键点模型，也优先选择本地浏览器模型，继续保持免费部署。
